== Description ==

A simple plugin to add a slideshow of images on the frontend using the [myslideshow] shortcode. Admin can upload and reorder images from the settings page.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the plugin 'Upload Images' to upload and reorder your images.
4. Use the [myslideshow] shortcode in any page, post, or widget to display the slideshow.

== Frequently Asked Questions ==

= How do I add images to the slideshow? =
Go to the plugin's settings page 'Upload Images', and you can upload images there.

= How do I reorder images in the slideshow? =
The images can be reordered by dragging and dropping them using the sortable interface in the plugin settings page.

= What happens if I delete an image from the settings page? =
The image will be removed from the slideshow on the frontend.
